# 📱 영어 단어 학습 앱 (englishWordApp)

> ** 한 줄 요약**  
> 사용자가 등록한 영어 단어를 효율적으로 암호화하고 퀴즈를 통해 학습할 수 있도록 돕는 안드로이드 토익단어장 프로젝트입니다.

---

## 📅 개발 기간
* 2025.07 ~ 2026.06 (소요, 계속 개발중)
* 개인 프로젝트 (기획, 디자인, 개발 100%)

## 🛠️ 사용 기술 및 개발 환경
* **Language:** Kotlin
* **IDE:** Android Studio
* **SDK Version:** Min 24 / Target 34
* **Architecture:** MVVM 패턴 (선택 사항)
* **Database:** Room DB 또는 SharedPreferences (본인이 쓴 것 적기)

## 📌 핵심 기능
* **단어장 관리:** 단어 추가, 수정, 삭제 기능 구현
* **퀴즈 모드:** 객관식/주관식 테스트 기능 및 점수 계산
* **통계 확인:** 틀린 단어만 모아보는 오답노트 기능
